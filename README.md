# Geo-quiz

## Simple example of a geagraphy quiz built with React

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/belka-mironova/mesto-react.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the application in a development mode
   ```sh
   npm start
   ```

* JavaScript
